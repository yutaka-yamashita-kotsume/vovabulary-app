### 本サービスの概要
英語学習者向けの単語学習アプリ
機能はシンプルにして、覚えたい単語を記録し、ランダムな簡易テストをし、記憶の定着を促す機能を実装する。
キャッチコピー
「広告ゼロ、ストレスゼロ。あなたの3分を、すべて学習に。」

想定ユーザー
通勤中でスマホで学習したい社会人

サービスの利用イメージ
シンプル操作でわからない英単語を記録する。その後、API連携により日本語訳と音声が自動で登録される。
また、簡易テスト(10問)を実施する。英単語を見て、日本語の意味を答える。（出題された英単語は発音が聞ける）
同じ単語で３回連続正解したら、その単語は定着済みになり、次回の簡易テストからは出題されないようなアプリにする。
また、英単語長自体にも日本語訳と発音、正解数を記録する。

サービスの差別化ポイント
 ①単語帳と簡易テストと学習記録に絞ることで、シンプルで使用しやすいこと。
 ②簡易テストにて、同じ単語で3問連続正解するまでランダムに出題されるため、英単語の習熟度が高い。
 ③広告を挟まないので、スムーズにアプリ内の動作ができること。ユーザーのストレスが無い。
 ④英単語登録は、英単語のみを登録するだけで、日本語訳と発音が自動登録されること。
 ④学習の進捗を確認できることで、学習のモチベーションが保てる。
 ⑤学習の進捗をXに共有することで、学習のモチベーションを促す。

## 機能候補
現状作ろうと思っている機能、案段階の機能をしっかりと固まっていなくても構わないのでMVPリリース時に作っていたい機能
→
・単語帳に英単語を記録する機能
L・英単語入力時、日本語を自動補完するAPI連携機能
 ・日本語訳が表示される機能
 ・発音が聞こえる機能
 ・正解数が記録される機能
・簡易テスト機能
　L・正解、不正解を記録する機能
   ・出題時に音声が聞こえる機能
・ログイン機能

本リリースまでに作っていたい機能
→・抜き打ち復習機能
・学習日数をカウントする機能
・３問連続正解した、単語数を画面に表示する機能。
・３問連続した英単語に、特別なアイコンをつける。（王冠など）
・今日の学習単語数をXでシェアする機能（「#英語垢」「#TOEIC」といったハッシュタグを付与する）
・APPstoreでの、アプリ配信

どこにユニークさや使いやすさがあるか？
→シンプルな設計、広告がないこと。Xで学習状況を報告できること。
