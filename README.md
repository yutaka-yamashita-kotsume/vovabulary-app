### 本サービスの概要
## サービス概要
英語学習者向けの単語学習アプリ
機能はシンプルにして、覚えたい単語を記録し、ランダムな簡易テストをし、記憶の定着を促す機能を実装したい。

キャッチコピー
「広告ゼロ、ストレスゼロ。あなたの3分を、すべて学習に。」

## このサービスへの思い・作りたい理由
紙の単語帳では、模擬試験や参考書内で分からなかった単語を一つに集約できなく、効率が悪いと感じた。また、持ち運びの手間や、発音がわからないので、リスニングに対応できないなどのデメリットもある。
さらに、既存アプリでは広告が入ったり、課金しないと単語帳の機能が利用できないことがあり不便であると感じたため、シンプルな単語帳のアプリが欲しいと感じた。

## ユーザー層について
通勤中でスマホで学習したい社会人

## サービスの利用イメージ
どんな機能・仕組みで解決するのか？
→シンプルにわからない英単語を記録する。その後、API連携により日本語訳が自動で登録される。
また、簡易テスト(10問)を実施する。英単語を見て、日本語の意味を答える。（出題された英単語は発音が聞ける）
同じ単語で３回連続正解したら、その単語は定着済みになり、次回の簡易テストからは出題されないようなアプリにする。
また、英単語長自体にも日本語訳と発音、正解数を記録する。

どこにユニークさや使いやすさがあるか？
→シンプルな設計、広告がないこと。Xで学習状況を報告できること。

似たようなサービスが存在する場合、そのサービスとの明確な差別化ポイントとその差別化ポイントのどこが優れているのか教えてください。
→①単語帳と簡易テストと学習記録に絞ることで、シンプルで使用しやすいこと。
 ②簡易テストにて、同じ単語で3問連続正解するまでランダムに出題されるため、英単語の習熟度が高い。

 ④学習の進捗を確認できることで、学習のモチベーションが保てる。
 ⑤学習の進捗をXに共有することで、学習のモチベーションを促す。

## 他社比較
Wordholic
強み
・自分の単語帳を作成できる
弱み
・ui/uxが使いづらい、ハンバーガーメニューから各機能を経由して利用するため、不便
・単語登録の際に、日本語と英語どちらも入力しないといけない
・発音機能はあるが、多言語になっているため、学習したい言語を設定するのが面倒
・広告がいろんなページにあるため、視認性と使い勝手が悪い

mikan 
強み
・ui/ux、デザインが良い
・英語学習者の目的に沿って、単語帳(教材)を自動セレクトされる
・学習データがグラフで見れる
弱み
・単語帳は指定されたものになるため、既に知っている単語も出題される
・わからない単語を登録する機能はあるが、無課金だと月20単語までしか登録できない

abceed
強み
・my単語帳を作成できる
・APIにより、英単語入力だけで、日本語訳と発音が登録される
弱み
・簡易テストで間違った問題を復習できるが、一回正解すると次回から出題されなくなる。（定着度が低いままになる）
・英単語の連続正解の記録がつけられない。
