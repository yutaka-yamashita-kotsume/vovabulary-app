## サービス概要
英語学習者向けの単語学習アプリ
機能はシンプルにして、覚えたい単語を記録し、簡易テストをし、定着を促す機能を実装したい。
キャッチコピー
「広告ゼロ、ストレスゼロ。あなたの3分を、すべて学習に。」

## このサービスへの思い・作りたい理由
紙の単語帳では、模擬試験や参考書内で分からなかった単語を一つに集約できなく、効率が悪いと感じた。また、持ち運びの手間や、発音がわからないので、リスニングに対応できないなどのデメリットもありました。
さらに、既存アプリでは広告が入ったり、課金しないと単語帳の機能が利用できないことがあり不便であると感じたため、シンプルな単語帳のアプリが欲しいと感じた。
