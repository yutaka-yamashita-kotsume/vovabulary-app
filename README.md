## サービス概要
英語学習者向けの単語学習アプリ
機能はシンプルにして、覚えたい単語を記録し、ランダムな簡易テストをし、記憶の定着を促す機能を実装したい。

キャッチコピー
「広告ゼロ、ストレスゼロ。あなたの3分を、すべて学習に。」

## このサービスへの思い・作りたい理由
紙の単語帳では、模擬試験や参考書内で分からなかった単語を一つに集約できなく、効率が悪いと感じた。また、持ち運びの手間や、発音がわからないので、リスニングに対応できないなどのデメリットもある。
さらに、既存アプリでは広告が入ったり、課金しないと単語帳の機能が利用できないことがあり不便であると感じたため、シンプルな単語帳のアプリが欲しいと感じた。

## ユーザー層について
通勤中でスマホで学習したい社会人

## サービスの利用イメージ
どんな機能・仕組みで解決するのか？
→シンプルにわからない英単語を記録する。その後、API連携により日本語訳が自動で登録される。
また、簡易テスト(10問)を実施する。英単語を見て、日本語の意味を答える。（出題された英単語は発音が聞ける）
同じ単語で３回連続正解したら、その単語は定着済みになり、次回の簡易テストからは出題されないようなアプリにする。
また、英単語長自体にも、日本語訳と発音、正解数を記録する。

どこにユニークさや使いやすさがあるか？
→シンプルな設計、広告がないこと。Xで学習状況を報告できること。
